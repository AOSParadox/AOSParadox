AOSParadox 1.0
==============

AOSParadox is a ROM based on [Codeaurora sources](https://www.codeaurora.org/cgit/quic/la).
This 1.0 is based on Android Lollipop 5.0 

The goal of this ROM is to bring an AOSP ROM with graphic customization.
